Development environment

Steps to setup the folders for development

Commands

Commands to follow step by step

  • Open powershell or terminal
  • Go to top level or home folder: cd \ or cd ~
  • Create the folder: mkdir videodrömm
  • Change current folder: cd videodrömm
  • Clone Cinder or openFramework inside this folder, see sections below
  • Setup Visual Studio Community 2015 or use your favourite IDE
  • Create another subfolder called videodrömm for the repos we are going to work in
  • Create the folder: mkdir videodrömm
  • Change current folder: cd videodrömm
  • Choose the repo(s) you want to use, for example let's use the assets repo:
  • Fork the assets repo: https://github.com/videodromm/assets.git
  • Clone your assets repo: git clone https://github.com/YOURGITHUBNAME/assets.git
  • Configure the link to the parent folder for future pull requests: git remote add upstream git://github.com/videodromm/assets.git
  • Before coding, get latest modifications from the parent: git fetch upstream and git merge upstream/master master
  • Do some coding
  • Commit and push to your repo
  • Do a pull request for review and integration in the parent repo

Cinder

If you use Cinder, checkout Cinder: git clone https://github.com/cinder/Cinder.git --recursive
Open Cinder.sln in the vc2013 with Visual Studio 2015
Compile it with Visual Studio 2015 using Batch build: check 32 bit Debug and Release versions
Create a test project with Tinderbox

openFrameworks

If you use openFrameworks, checkout openFrameworks: git clone https://github.com/openframeworks/openFrameworks.git --recursive
Create a test project with projectGenerator

Python

If you use Python, setup version 3.5
